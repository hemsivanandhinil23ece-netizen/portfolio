Alcohol Consumption Detection

Duration: 3 months (Jan 2024 - Mar 2024)

Technologies: Arduino, MQ3 Alcohol Sensor, GSM Module, GPS

Description: This project aims to prevent drunk driving accidents by developing an IoT-based system that detects alcohol consumption in vehicle drivers. The system uses an MQ3 alcohol sensor to detect alcohol levels in the driver's breath. If the alcohol level exceeds a certain threshold, the system prevents the vehicle from starting. Additionally, it sends an SMS alert with GPS coordinates to emergency contacts.

Key Features:

  • Real-time alcohol detection using MQ3 sensor
  • Ignition control system integration
  • SMS alert system with location tracking
  • Data logging for analytical purposes

Outcome: Successfully implemented the prototype with 95% detection accuracy. The project was demonstrated at a road safety awareness program organized by the college.

Bone Fracture Recovery Monitoring

Duration: 4 months (Aug 2023 - Nov 2023)

Technologies: Raspberry Pi, Strain Gauges, Biomedical Sensors, Python

Description: This project focuses on creating a wearable device that monitors bone healing progress in real-time. The system uses strain gauges and biomedical sensors to track bone density, alignment, and stress distribution. Data is processed using machine learning algorithms to predict recovery time and detect anomalies in the healing process.

Key Features:

  • Non-invasive bone healing monitoring
  • Real-time data visualization through a mobile app
  • Personalized recovery progress reports
  • Alert system for abnormal healing patterns

Outcome: Developed a functioning prototype that was tested in collaboration with a local orthopedic clinic. The project received positive feedback from healthcare professionals for its potential application in remote patient monitoring.
